# Screenshots

Add your screenshots here:

- `home.png` - Home page with featured content (recommended: 1600x900px)
- `search.png` - Search page with filters (recommended: 1600x900px)

## How to Add Screenshots

1. Take screenshots of your application
2. Save them as `home.png` and `search.png` in this directory
3. Commit and push:
   ```bash
   git add screenshots/*.png
   git commit -m "Add screenshots"
   git push
   ```

The README will automatically display them!
